Minimal math notation
Minimal math notation
2022-03-16 00:00

With backquote

  • using $
    • This is $ \int_\epsilon^\pi f(x) $ some inline text!
  • using $$
    • $$ \int_\epsilon^\pi f(x) $$
  • using \(
    • This is \( \int_\epsilon^\pi f(x) \) some inline text!
  • using \[
    • \[ \int_\epsilon^\pi f(x) \]

Without backquote

  • using $
    • This is $ \int_\epsilon^\pi f(x) $ some inline text!
  • using $$
    • $$ \int_\epsilon^\pi f(x) $$
  • how about: z^{y _{x_ 2}} and using $$
    • $$ z^{y {x 2}} $$
  • using \(
    • This is ( \int_\epsilon^\pi f(x) ) some inline text!
  • using \[
    • [ \int_\epsilon^\pi f(x) ]

Testing of z^{y _{x_ 2}}

  • with nothing
    • z^{y {x 2}}
  • with backquotes
    • z^{y _{x_ 2}}
  • with $$ and backquote
    • \( z^{y _{x_ 2}} \)
  • with $$ no backquote
    • $$ z^{y {x 2}} $$

results in:

$$\newcommand\distr{P}$$ $$ \distr (Y|X) $$
